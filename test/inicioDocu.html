Descargar la Librería

Podemos añadir la librería a través de los gestores de paquetes de NodeJs a través de los comandos que muestran a continuación.

npm install body-ui
yarn add body-ui

Copiar y pegar este código en tu proyecto para añadir la librería a la página web de tu proyecto.
